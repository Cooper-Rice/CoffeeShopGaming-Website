# CoffeeShopGaming Website

Official website code for the CoffeeShopGaming website.

Built with plain HTML, CSS, and JavaScript. No frameworks, no build steps.

## Pages

- `index.html` - Home page with server info and team
- `seasons.html` - Information about past, present, and future seasons
- `library.html` - Picture gallery
- `contact.html` - Contact form (opens Gmail)

## Files

```
index.html
seasons.html
library.html
contact.html
styles.css
theme.js
hamburger.js
images/
```

## Features

- Dark/light mode
- Hamburger menu on mobile
- Contact form that opens Gmail compose
- No dependencies
