Welcome to the CoffeeShopGaming website

CoffeeShopGaming was originally founded as a minecraft server for people from a indie-game discord server, We eventually grew and rebranded into the CoffeeShopGaming you know today, a tiny but vibrant community full of amazing people.


The server

The current season is season 2, it is a forge 1.20.1 modpack centered around create mod and origins(++), the server has a minigame area with mingames like spleef & tnt run and more to come! we also aim to host bi-weekly events, our last events were the end fight and a custom parkour for a prize! check out the picture library to see some pictures from the server.
